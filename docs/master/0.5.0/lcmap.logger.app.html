-main

(-main & args)

This is the entry point. Note, however, that the system components are defined in lcmap.logger.components. In particular, lcmap.logger.components.system brings together all the defined (and active) components; that is the module which is used to bring the system up when (component/start …) is called.

‘lein run’ will use this as well as ‘java -jar’.
